배달긱이 최저가인 이유

배달긱은 일반적으로 흔히 볼 수 있는 한건 배달이 아니라 캠퍼스 내 ‘묶음 배달’을 원칙으로 합니다.

예약제 단체주문 방식으로 주문・배달 과정을 줄여 중간 용역비 및 배달 수수료를 절감함은 물론, 자체 알고리즘을 통해 정해진 시간 내 배달을 100% 보장합니다.

또한 업계 최저 수수료 정책을 유지하며, 누적 주문금액을 기준한 사장님용 배달 수수료를 통해 고객에게 배달 최저가를 제공하면서도 입점 사장님들까지 고려한 수익구조를 만들었습니다.

유의사항
  • 최저가 보상제란 배달긱에서 결제한 금액이 다른 배달앱에서 누구나 주문가능한 조건으로 주문한 금액보다 비싼 경우 그 차액만큼 배달긱 포인트로 200% 보상하는 프로모션
  • 배달긱 회원 대상으로, 배달완료일로부터 7일 이내 카카오톡 채널로 신청가능
    신청대상 및 신청기간이 아닌 경우 보상 불가
  • 최저가 보상 기준으로 면밀한 검토 과정을 거쳐 보상 여부를 판단하며, 검증 결과에 따라 정상가로 보이지 않을 경우 최저가 보상 신청이 받아들여지지 않을 수 있음
  • 최저가 보상 기준 안내
    상점 : 사업자번호까지 동일한 상점만 보상 가능
    주문메뉴 : 메뉴구성이 동일해야하며, 본인의 주문과 추가옵션까지 같아야 보상 가능
    최소주문금액 : 주문메뉴가 최소주문금액을 만족하지 않을 경우, 차액만큼 추가
    배달비 : 주문한 수령장소를 기준으로 책정
    주문금액 : 누구나 주문 가능한 가격 기준
    (특정회원 할인, 쿠폰•이벤트•프로모션 할인, 간편결제 또는 특정카드 할인 등 제외)
  • 최저가 보상신청 후 영업일 7일(월~금, 09시 ~ 18시) 이내 확인이 이루어지며, 신청결과는 카카오톡으로 안내됨
    신청 이후 배달긱 고객센터에서 확인하는 시점에서 해당 메뉴의 가격이 검증되지 않으면 보상되지 않음
  • 최저가 보상이 확정되면, 신청결과 안내일로부터 2일 이내 배달긱 포인트가 지급됨
  • 최저가 보상 포인트 안내
    보상금액 : 결제 차액의 200%
    (최대 보상금액은 건당 3만원을 넘지 않으며, ID별로 월 1회 보상 지급)
    지급된 포인트는 마이배달긱 > 포인트에서 ‘최저가 보상’이라는 이름으로 확인가능함
    최저가 보상 포인트는 지급일로부터 60일간 유효하며, 이후 소멸됨
  • 최저가 보상 예시
    배달긱에서 주문한 메뉴의 조건
    메뉴가격 10,000원 / 옵션가 4,000원 / 포인트 400원 / 최소주문금액 0원 / 배달비 0원
    타 배달앱 동일한 상점에서 주문메뉴와 동일한 메뉴의 배달 조건
    메뉴가격 9,000원 / 옵션가 3,000원 / 최소주문금액 8,900원 / 배달비 1,200원
    보상지급액
    = {(10,000원+4,000원-400원)} - (9,000원+3,000원+1,200원)} x 200%
    = (13,600원-13,200원) x 200%
    = 800원이 배달긱 포인트로 지급
